\# AgroSense 🌱  

Intelligent Soil Monitoring Platform using ESP32 + Flask



\---



\## 🔧 Hardware Used

\- ESP32

\- Soil Moisture Sensor

\- DHT11 (Temperature \& Humidity)

\- LDR (Light Sensor)

\- Breadboard

\- Jumper Wires



\---



\## 💻 Software \& Tools

\- Arduino IDE

\- Python (Flask)

\- SQLite

\- DB Browser for SQLite

\- Chart.js (Dashboard Visualization)



\---



\## ⚙️ Features

\- Real-time monitoring of soil moisture, temperature, humidity, and light

\- ESP32 sends sensor data to Flask server using HTTP (JSON)

\- Data stored in SQLite database

\- Live dashboard with graphs and analytics

\- Smart plant health status detection



\---



\## 🔄 Data Flow

ESP32 → Flask API → SQLite Database → Web Dashboard
